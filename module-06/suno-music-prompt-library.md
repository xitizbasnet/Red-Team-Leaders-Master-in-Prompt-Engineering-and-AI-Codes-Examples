## How to Use This Library

Each prompt includes:
- **Style tags** for the Suno style field
- **Song structure** with lyrics (for Custom Mode) or description (for Simple Mode)
- **Notes** on what to expect and how to adjust

**Key:** When using Custom Mode, paste the style tags into the "Style of Music" field and the lyrics into the "Lyrics" field.

## Tips for Better Suno Results

1. **Be specific with sub-genres**: "Midwest emo" is better than "rock"
2. **Include BPM**: Helps set the right tempo expectations
3. **Name instruments**: The AI responds well to specific instrument mentions
4. **Reference artists/bands**: Helps define the sound direction
5. **Keep lyrics singable**: Match syllable counts across verses
6. **Use structure tags**: [Verse], [Chorus], [Bridge] etc. guide the arrangement
7. **Describe production style**: "Lo-fi bedroom" vs. "polished studio" changes everything
8. **Generate multiple versions**: Create 6-10 and pick the best
9. **Use parenthetical notes**: (soft), (building), (powerful) guide dynamics
10. **Keep it focused**: Better to nail one genre than blend too many poorly
