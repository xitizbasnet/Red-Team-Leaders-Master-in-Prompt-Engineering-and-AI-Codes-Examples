# Midjourney Prompt Library: 100+ Categorized Prompts

## Module 06: Code and Examples

---

## How to Use This Library

Each prompt is ready to use with `/imagine prompt:` in Midjourney. Parameters are included and can be adjusted to taste. Prompts are organized by category for easy reference.

**Legend:**
- `--ar` = Aspect Ratio
- `--s` = Stylize (0-1000)
- `--c` = Chaos (0-100)
- `--w` = Weird (0-3000)
- `--style raw` = Less Midjourney aesthetic, more literal

## Usage Tips

1. **Copy and paste directly** into Midjourney after `/imagine prompt:`
2. **Adjust --s (stylize)** up for more artistic interpretation, down for more literal results
3. **Add --style raw** for more photographic, less Midjourney-aesthetic results
4. **Change --ar** to match your target platform dimensions
5. **Add --c (chaos)** for more varied results in the initial grid
6. **Combine prompts** by taking elements from different categories
7. **Use --sref** with a generated image URL to maintain style consistency
8. **Use --cref** with a character image URL for character consistency
9. **Iterate**: Generate, select the best direction, refine the prompt, and regenerate
10. **Save seeds** for results you want to revisit or create variations of
