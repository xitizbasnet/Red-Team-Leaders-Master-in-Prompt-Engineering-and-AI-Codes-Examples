# Creative Workflow Checklists for AI Projects

## Module 06: Code and Examples

---

## How to Use These Checklists

Each checklist provides a step-by-step workflow for a specific type of creative project using AI tools. Follow the steps in order, checking off each item as you complete it. Estimated time ranges are provided for each phase.

---

## Checklist 1: Social Media Content Pack (7 Posts)

**Estimated Total Time: 3-5 hours**

### Phase 1: Planning (30-45 min)
```
□ Define content goals and key messages for the week
□ Choose themes/topics for each of the 7 posts
□ Determine platforms (Instagram, Twitter/X, LinkedIn, etc.)
□ Note required aspect ratios for each platform
□ Define visual style and brand consistency anchors
□ Write draft captions for all 7 posts
□ Identify which posts need images, video, or carousels
```

### Phase 2: Visual Generation (60-90 min)
```
□ Create prompt template with brand style suffix
□ Generate 3-4 options per post using AI image tool
  □ Post 1: [Topic] — Generated □  Selected □
  □ Post 2: [Topic] — Generated □  Selected □
  □ Post 3: [Topic] — Generated □  Selected □
  □ Post 4: [Topic] — Generated □  Selected □
  □ Post 5: [Topic] — Generated □  Selected □
  □ Post 6: [Topic] — Generated □  Selected □
  □ Post 7: [Topic] — Generated □  Selected □
□ Select best image for each post
□ Upscale selected images to required resolution
□ Check for AI artifacts in all selected images
```

### Phase 3: Editing and Formatting (45-60 min)
```
□ Crop/resize images for each target platform
  □ Feed posts (1:1 or 4:5)
  □ Stories/Reels (9:16)
  □ Twitter/LinkedIn (16:9 or 1.91:1)
□ Add text overlays if needed (brand font, colors)
□ Create carousel slides if applicable
□ Generate any short video clips (3-5 sec for Reels)
□ Apply consistent color grading across all images
□ Add brand watermark or logo if required
```

### Phase 4: Copy and Finalization (30-45 min)
```
□ Finalize all captions (check grammar, tone, length)
□ Add relevant hashtags for each platform
□ Write alt text for accessibility
□ Review all assets against brand guidelines
□ Quality check: all formats, sizes, and quality correct
□ Export final files in platform-optimized formats
□ Schedule posts in social media management tool
□ Document prompts and settings used (for future reference)
```

### Quality Gate
```
□ All images free of AI artifacts?
□ Brand colors consistent across all posts?
□ Visual style cohesive (looks like one campaign)?
□ Text overlays readable and properly positioned?
□ Captions appropriate for each platform?
□ File sizes optimized for web delivery?
□ AI disclosure added where required by platform?
```

## Checklist 2: Product Photography Set (10 Images)

**Estimated Total Time: 4-6 hours**

### Phase 1: Shot Planning (30 min)
```
□ Define the product and its key features
□ List all required shots:
  □ Hero shot (main marketing image)
  □ Front view
  □ Side view
  □ Back view
  □ 3/4 angle view
  □ Detail/macro shots (2-3)
  □ Lifestyle/context shot
  □ Flat lay / top-down view
  □ Scale reference shot
□ Define background and styling preferences
□ Define lighting style (bright/minimal or dark/moody)
□ Gather reference images for style direction
```

### Phase 2: Prompt Development (30 min)
```
□ Create base prompt template with consistent elements:
  □ Product description
  □ Lighting setup description
  □ Background/surface description
  □ Quality/style tags
  □ Camera/lens specifications
□ Write specific prompts for each shot angle
□ Define parameter settings (aspect ratio, stylize, etc.)
□ Create negative prompt (if using Stable Diffusion)
```

### Phase 3: Generation (60-90 min)
```
□ Generate hero shot (8-10 variations, select best 2)
□ Generate front view (4-6 variations, select best)
□ Generate side view (4-6 variations, select best)
□ Generate back view (4-6 variations, select best)
□ Generate 3/4 view (4-6 variations, select best)
□ Generate detail shots (4-6 each, select best)
□ Generate lifestyle shot (6-8 variations, select best)
□ Generate flat lay (4-6 variations, select best)
□ Generate scale reference (4-6 variations, select best)
□ Review all selections for consistency as a set
```

### Phase 4: Post-Processing (60-90 min)
```
□ Upscale all selected images (Topaz or Magnific)
□ Inpaint any AI artifacts or imperfections
□ Remove background where clean cutouts needed
□ Apply consistent color grading across all images
□ Ensure product color accuracy
□ Add natural shadows for cut-out images
□ Sharpen for web or print as needed
□ Export in required formats:
  □ High-res PNG (master files)
  □ E-commerce size (typically 2000x2000px)
  □ Web-optimized JPEG
  □ Thumbnail size
```

### Quality Gate
```
□ Product appears identical across all angles?
□ Colors accurate and consistent?
□ Lighting direction consistent?
□ No AI artifacts on the product itself?
□ Background clean and appropriate?
□ Resolution meets e-commerce requirements?
□ File naming convention applied?
□ Images could pass as real product photography?
```

## Checklist 3: Promotional Video (30 seconds)

**Estimated Total Time: 6-10 hours**

### Phase 1: Pre-Production (60-90 min)
```
□ Define video objective and key message
□ Identify target audience and platform(s)
□ Write video script (narration and/or text overlays)
□ Create storyboard (sketch or describe each shot)
  □ Shot 1: Description / Duration / Camera
  □ Shot 2: Description / Duration / Camera
  □ Shot 3: Description / Duration / Camera
  □ Shot 4: Description / Duration / Camera
  □ Shot 5: Description / Duration / Camera
  □ Shot 6: Brand card / Logo / CTA
□ Define visual style anchors (mood, colors, lighting)
□ Plan music style and voiceover requirements
□ Determine aspect ratios needed (16:9, 9:16, 1:1)
```

### Phase 2: Asset Generation (120-180 min)
```
Video Clips:
□ Generate key frame images for each shot (Midjourney/DALL-E)
□ Generate video clip 1 (4-6 versions, select best)
□ Generate video clip 2 (4-6 versions, select best)
□ Generate video clip 3 (4-6 versions, select best)
□ Generate video clip 4 (4-6 versions, select best)
□ Generate video clip 5 (4-6 versions, select best)
□ Review clip consistency (style, color, quality)

Audio:
□ Generate background music with Suno (6-10 versions)
□ Select best music track
□ Trim and edit music to 30-second duration
□ Generate voiceover with ElevenLabs
□ Adjust voiceover pacing and timing
□ Export clean audio files (WAV format)
```

### Phase 3: Assembly and Editing (90-120 min)
```
□ Import all clips into video editor (Premiere/DaVinci/CapCut)
□ Arrange clips in storyboard order
□ Trim clips to exact timing
□ Add transitions between clips (dissolve, cut, etc.)
□ Sync voiceover to corresponding clips
□ Add background music and adjust levels
□ Add text overlays and lower thirds
□ Add logo animation and end card
□ Add call-to-action text/button visual
□ Color grade all clips for consistency
□ Review pacing and rhythm
```

### Phase 4: Finalization (30-60 min)
```
□ Watch full video 3 times, noting any issues
□ Fix any timing, transition, or quality issues
□ Ensure audio levels are balanced (voice over music)
□ Add captions/subtitles
□ Export in required formats:
  □ 16:9 horizontal (YouTube, website)
  □ 9:16 vertical (Instagram Reels, TikTok, Stories)
  □ 1:1 square (Instagram feed, LinkedIn)
□ Optimize file size for each platform
□ Test playback on mobile device
□ Create thumbnail image from best frame
```

### Quality Gate
```
□ Video tells a clear, compelling story in 30 seconds?
□ Visual consistency maintained across all clips?
□ No obvious AI artifacts visible during playback?
□ Audio (voice, music) clear and properly mixed?
□ Text is readable on mobile screens?
□ Brand elements (logo, colors, voice) present and correct?
□ Call-to-action is clear?
□ Multiple aspect ratios exported and verified?
□ Captions/subtitles accurate?
□ AI content disclosure included if required?
```

## Checklist 4: Brand Identity Package

**Estimated Total Time: 8-12 hours**

### Phase 1: Strategy and Research (60-90 min)
```
□ Define brand mission, vision, and values
□ Identify target audience and demographics
□ Research competitor visual identities
□ Define brand personality (3-5 adjective descriptors)
□ Choose brand voice tone (formal, casual, playful, etc.)
□ Determine deliverables needed:
  □ Logo concepts
  □ Color palette
  □ Typography direction
  □ Mood board
  □ Brand imagery set
  □ Social media templates
  □ Brand guidelines document
```

### Phase 2: Visual Direction (90-120 min)
```
□ Generate mood board images (8-12 images across:)
  □ Color/texture inspiration (2-3 images)
  □ Photography style reference (2-3 images)
  □ Typography and layout inspiration (2-3 images)
  □ Overall mood and atmosphere (2-3 images)
□ Define color palette:
  □ Primary colors (2-3) with hex codes
  □ Secondary colors (2-3) with hex codes
  □ Accent color (1) with hex code
  □ Neutrals (light and dark) with hex codes
□ Generate logo concepts (3 different approaches)
  □ Symbol/icon approach
  □ Typography approach
  □ Combination mark approach
□ Select top direction for each element
```

### Phase 3: Asset Generation (120-180 min)
```
□ Refine selected logo concept (variations and colorways)
□ Generate brand imagery set (8-12 images):
  □ Hero/banner images
  □ Lifestyle images
  □ Product context images
  □ Abstract/background textures
  □ People/team representation
  □ Environment/location images
□ Upscale all images to high resolution
□ Generate social media template backgrounds (3-5 styles)
□ Create consistent prompt templates for future generation
```

### Phase 4: Documentation (60-90 min)
```
□ Compile brand guidelines document:
  □ Brand story and values
  □ Logo usage rules (sizes, spacing, what not to do)
  □ Color palette with values (hex, RGB, CMYK)
  □ Typography guidelines
  □ Photography/imagery style guide
  □ Tone of voice guide
  □ AI prompt templates for consistent generation
  □ Social media specifications
□ Organize all assets in logical folder structure
□ Name all files with consistent naming convention
□ Create README file explaining the package contents
```

### Quality Gate
```
□ All brand elements feel cohesive?
□ Logo works at small and large sizes?
□ Color palette is harmonious and distinctive?
□ Imagery is consistently styled?
□ Brand personality comes through in all elements?
□ Guidelines are clear and actionable?
□ AI prompts are documented for reproducibility?
□ All files are high-resolution and properly formatted?
□ Package would be presentable to a client?
```

## Checklist 5: Podcast Episode Production

**Estimated Total Time: 3-5 hours**

### Phase 1: Content Preparation (45-60 min)
```
□ Define episode topic and key takeaways
□ Research topic and gather key points
□ Write episode script or detailed outline:
  □ Introduction (hook, topic intro, what listener will learn)
  □ Main content sections (3-5 sections)
  □ Transitions between sections
  □ Conclusion and call-to-action
□ Format script for TTS optimization:
  □ Numbers written as words
  □ Abbreviations expanded
  □ Pronunciation guides for unusual terms
  □ Pause markers at section breaks
  □ Emphasis markers for key points
□ Estimate total episode duration
```

### Phase 2: Audio Generation (60-90 min)
```
□ Select or design narrator voice (ElevenLabs)
□ Configure voice settings:
  □ Stability: [value]
  □ Similarity: [value]
  □ Style: [value]
□ Generate episode audio:
  □ Intro segment — Generated □  Reviewed □  Approved □
  □ Section 1 — Generated □  Reviewed □  Approved □
  □ Section 2 — Generated □  Reviewed □  Approved □
  □ Section 3 — Generated □  Reviewed □  Approved □
  □ Conclusion — Generated □  Reviewed □  Approved □
□ Regenerate any segments with pronunciation issues
□ Generate intro/outro music jingle (Suno)
□ Generate transition sound effects if needed
```

### Phase 3: Editing and Assembly (45-60 min)
```
□ Import all audio segments into editor (Audacity/Adobe Audition)
□ Arrange segments in correct order
□ Add intro music and fade-in
□ Add transitions between sections
□ Insert transition sounds if applicable
□ Add outro music and fade-out
□ Normalize audio levels across all segments
□ Apply subtle compression for consistent volume
□ Apply EQ for clarity (if needed)
□ Add room tone/ambient background (if desired)
□ Export master file (WAV, 44.1kHz, 16-bit)
```

### Phase 4: Publishing Preparation (30-45 min)
```
□ Export in podcast format (MP3, 128kbps or higher)
□ Add ID3 tags (title, description, artwork, episode number)
□ Generate episode cover art:
  □ Create episode-specific image (DALL-E/Midjourney)
  □ Add episode number and title text
  □ Export at 3000x3000 minimum (square)
□ Write episode description for podcast platforms
□ Create show notes with timestamps
□ Generate transcript (for accessibility)
□ Upload to podcast hosting platform
□ Create social media promotion post:
  □ Audiogram clip (30-60 seconds of highlights)
  □ Quote graphic from episode
  □ Episode announcement post
```

### Quality Gate
```
□ Audio sounds natural and professional?
□ No mispronunciations or garbled words?
□ Volume consistent throughout?
□ Music and speech levels properly balanced?
□ Transitions smooth and not jarring?
□ Episode flows logically from intro to conclusion?
□ Duration matches target length?
□ File format meets platform requirements?
□ Cover art is clear and readable at small size?
□ Show notes and description are complete?
```

## Checklist 6: Illustrated eBook / Guide

**Estimated Total Time: 12-20 hours (for 20-page guide)**

### Phase 1: Content and Structure (120-180 min)
```
□ Define guide topic, audience, and goals
□ Create detailed outline with chapters/sections
□ Write all text content (or generate with AI assistance)
□ Identify illustration needs per section:
  □ Full-page illustrations
  □ Spot illustrations (small, inline)
  □ Diagrams or infographics
  □ Header/chapter opener images
  □ Background textures/patterns
□ Create illustration brief for each image
□ Define overall visual style (consistent across all illustrations)
□ Choose page layout and dimensions
```

### Phase 2: Visual Style Development (60-90 min)
```
□ Generate 10-15 style exploration images
□ Select winning visual style
□ Create style reference image (for --sref or IP-Adapter)
□ Test style consistency with 3-4 diverse subjects
□ Define prompt template with consistent style descriptors
□ Create color palette for illustrations
□ Lock character designs (if characters appear throughout)
□ Document all style decisions and prompt formulas
```

### Phase 3: Illustration Generation (180-300 min)
```
For each illustration needed:
  □ Write specific prompt based on section content
  □ Generate 4-8 options
  □ Select best match for the content
  □ Upscale to print resolution (300 DPI at target size)
  □ Inpaint any issues
  □ Apply consistent color correction

Track progress:
  □ Illustration 1: Generated □  Enhanced □  Approved □
  □ Illustration 2: Generated □  Enhanced □  Approved □
  □ Illustration 3: Generated □  Enhanced □  Approved □
  [Continue for all illustrations...]

□ Review all illustrations together for consistency
□ Make color/style adjustments for any outliers
```

### Phase 4: Layout and Assembly (120-180 min)
```
□ Import text and images into layout tool (Canva, InDesign, etc.)
□ Set up page templates and master pages
□ Place text content on all pages
□ Place illustrations in correct positions
□ Adjust text flow around images
□ Add headers, footers, and page numbers
□ Add table of contents
□ Add cover page (full illustration + title)
□ Add back cover and any appendix content
□ Review typography (consistency, readability)
□ Review alignment and spacing
□ Proofread all text content
```

### Phase 5: Export and Distribution (30-45 min)
```
□ Export print-ready PDF (300 DPI, CMYK if printing)
□ Export screen-optimized PDF (RGB, smaller file)
□ Export individual pages as images if needed
□ Create social media preview images
□ Test PDF on multiple devices (desktop, tablet, phone)
□ Create landing page or distribution plan
□ Prepare promotional materials
```

### Quality Gate
```
□ All illustrations match the defined visual style?
□ Text is proofread and error-free?
□ Layout is consistent across all pages?
□ Images are high enough resolution for target output?
□ Colors are accurate (especially for print)?
□ Table of contents is accurate?
□ Guide flows logically from start to finish?
□ Cover is eye-catching and professional?
□ File size is appropriate for distribution method?
□ AI disclosure included if required?
```
