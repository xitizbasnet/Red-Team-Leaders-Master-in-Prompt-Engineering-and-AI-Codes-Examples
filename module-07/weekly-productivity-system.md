```

Name: [YOUR FULL NAME]
Role: [YOUR JOB TITLE]
Company: [COMPANY NAME] — [WHAT THE COMPANY DOES IN ONE SENTENCE]
Department: [YOUR DEPARTMENT]
Reports to: [YOUR MANAGER'S NAME AND TITLE]
Direct reports: [NUMBER AND THEIR ROLES, OR "NONE"]

Current projects:
1. [PROJECT NAME] — Status: [ON TRACK / AT RISK / BEHIND] — Deadline: [DATE]
2. [PROJECT NAME] — Status: [___] — Deadline: [___]
3. [PROJECT NAME] — Status: [___] — Deadline: [___]

Quarterly goals (current quarter):
1. [GOAL 1 — MEASURABLE]
2. [GOAL 2 — MEASURABLE]
3. [GOAL 3 — MEASURABLE]

Working hours: [START] to [END], [TIME ZONE]
Peak productivity hours: [RANGE]

- Email: [FORMAL / FRIENDLY-PROFESSIONAL]
- Slack: [CASUAL / PROFESSIONAL]

- [E.G., "Too many meetings on Wednesdays"]
- [E.G., "Struggle with delegation"]
- [E.G., "Procrastinate on report writing"]

- Default to bullet points and tables

[LIST ANY CARRYOVER TASKS]

   (Heavy meetings? Big deadline? Quiet and strategic?)

   (Keep this to 30 minutes maximum — do not work the weekend)

- [WIN 1]
- [WIN 2]

- [CHALLENGE 1]
- [CHALLENGE 2]

Waiting on others: [NAME — WHAT — SINCE WHEN]
New items to add: [ANYTHING NEW THAT CAME UP]

MY QUARTERLY GOALS (REMINDER):
1. [GOAL 1]
2. [GOAL 2]
3. [GOAL 3]

Energy and context:
- Expected energy level: [HIGH / NORMAL / LOW]
- Any personal commitments affecting the week: [IF ANY]
- Best deep work days this week: [DAYS]

1. Top 3 priorities for this week (aligned to quarterly goals)
2. "Big Rocks" — the 2-3 tasks that define a successful week
3. Day-by-day task allocation:
   Monday: [TASKS + GOALS FOR THE DAY]
   Tuesday: [...]
   Wednesday: [...]
   Thursday: [...]
   Friday: [...]
4. Tasks to delegate (and to whom)
5. Tasks to defer to next week (and why it is okay)
6. One proactive item (something important but not urgent to advance)
7. Risk assessment: What could derail this week?
8. Weekly habit focus: [ONE HABIT TO MAINTAIN THIS WEEK]
```

### Prompt M2: Monday Morning Email Triage

```
Monday morning email triage. Categorize these emails from the weekend
and this morning:

1. FROM: [sender] | SUBJECT: [subject] | PREVIEW: [first line]
2. FROM: [sender] | SUBJECT: [subject] | PREVIEW: [first line]
3. FROM: [sender] | SUBJECT: [subject] | PREVIEW: [first line]
[CONTINUE FOR ALL UNREAD]

Categorize each as:
- NOW (respond within 2 hours)
- TODAY (respond before end of day)
- THIS WEEK (respond within the week)
- DELEGATE (suggest who should handle)
- FYI (read, no response needed)
- ARCHIVE (no action needed)

For NOW and TODAY items, suggest a one-sentence response strategy.
For DELEGATE items, draft a forwarding message.
```

## DAILY PROMPTS: Use Every Workday

### Prompt D1: Morning Kickoff (5 minutes — every morning)

```
Good morning. Here is my context for today:

Today is: [DATE, DAY]
Today's plan from weekly planning: [PASTE TODAY'S GOALS FROM MONDAY'S PLAN]
Calendar today:
[LIST ALL MEETINGS WITH TIMES]

Yesterday's results:
- Completed: [WHAT I FINISHED]
- Not completed: [WHAT CARRIED OVER]
- New items that came up: [ANYTHING ADDED]

Energy level today: [1-10]
Available deep work time: [HOURS]

Please:
1. Confirm or adjust today's priorities
2. Create a time-blocked schedule
3. Flag any conflicts or concerns
4. If energy is low (below 6), suggest a lighter plan
5. One motivational nudge based on this week's progress
```

### Prompt D2: Pre-Meeting Quick Prep (2 minutes — before each meeting)

```
Quick prep — meeting in 10 minutes.

Meeting: [NAME]
With: [KEY ATTENDEES]
Topic: [SUBJECT]
My goal: [WHAT I WANT FROM THIS MEETING]
Duration: [MINUTES]

Give me:
1. My opening statement (one sentence to set the tone)
2. Two questions to ask
3. One thing to watch out for (potential issue or objection)
```

### Prompt D3: Post-Meeting Action Items (3 minutes — after each meeting)

```
Process these meeting notes:

Raw notes:
[PASTE YOUR MESSY NOTES]

Output:
1. Summary (3 sentences max)
2. Decisions made:
   - [DECISION 1]
   - [DECISION 2]
3. Action items:
   | Action | Owner | Due | Priority |
   |--------|-------|-----|----------|
4. Follow-up needed: [ANY OPEN QUESTIONS]
5. Draft follow-up email (under 100 words — send to attendees)
```

```

[SAME FORMAT]

EMAIL 3:
[SAME FORMAT]

4. Set up tomorrow's preliminary plan (top 3 priorities)
5. Note if I am falling behind on any weekly goals
```

## TUESDAY: Deep Work and Research Day

### Prompt T1: Deep Work Planner

```
I have [X] hours of uninterrupted deep work time today.

My most important deep work task: [DESCRIPTION]
Secondary task if time allows: [DESCRIPTION]

Help me structure this time:
1. Break the main task into specific 25-minute Pomodoro segments
   (what to focus on in each segment)
2. Define "done" — what does completion look like?
3. What do I need to have ready before I start? (files, tools, references)
4. One thing to avoid that typically distracts me during deep work
5. A "good enough" version if I run out of time
   (the minimum viable output)
```

### Prompt T2: Research Synthesis

```
I have been gathering research on [TOPIC]. Help me synthesize.

My findings:
[PASTE ALL YOUR NOTES, DATA, AND FINDINGS]

Create:
1. Executive summary (one paragraph)
2. Key findings by theme (not by source)
3. Data points that support the main conclusion
4. Contradictions or uncertainties
5. Gaps in my research (what I still need to find out)
6. Recommended next steps
7. "Bottom line" statement for decision-makers
```

## WEDNESDAY: Meetings and Communication Day

### Prompt W1: Meeting-Heavy Day Strategy

```
Today is a heavy meeting day. I have [NUMBER] meetings.

Meetings:
[LIST ALL WITH TIMES, DURATIONS, AND TOPICS]

Free time between meetings: [LIST THE GAPS]

Help me:
1. Identify which meetings I must be fully engaged in vs. which I can
   listen passively
2. Batch-prepare for all meetings (use the meeting prep format)
3. Plan what to do in the gaps between meetings (realistic tasks for
   short windows)
4. Identify one meeting I could potentially skip, shorten, or delegate
5. Suggest an end-of-day decompression activity (meetings are draining)
```

### Prompt W2: Communication Batch

## Emergency / Ad-Hoc Prompts

### When Overwhelmed

```
I am feeling overwhelmed right now. Here is everything on my plate:
[BRAIN DUMP EVERYTHING — DO NOT ORGANIZE, JUST LIST]

Help me:
1. Separate urgent from important
2. Identify what can wait until next week
3. Identify what I can delegate or say no to
4. Find the ONE thing that would give me the most relief right now
5. Create a realistic plan for the rest of today (not the rest of the week)
6. Give me a 2-minute breathing/reset exercise before I start
```

### When Stuck on a Decision

```
I need to make a decision and I am going in circles.

Decision: [WHAT I NEED TO DECIDE]
Options: [LIST OPTIONS]
Deadline: [WHEN I NEED TO DECIDE BY]
Stakeholders: [WHO IS AFFECTED]
My gut feeling: [WHAT MY INSTINCT SAYS]
What is holding me back: [WHY I CANNOT DECIDE]

Help me:
1. List the pros and cons of each option objectively
2. Identify what information I am missing (if any)
3. Apply the "10-10-10 rule": How will I feel about this decision in
   10 minutes? 10 months? 10 years?
4. What would I tell a friend in this situation?
5. What is the cost of NOT deciding?
6. Recommend a decision with your reasoning
```

### When Starting Something From Scratch

```
I need to start a new [PROJECT / TASK / INITIATIVE] and I do not know
where to begin.

What I need to do: [DESCRIPTION]
Deadline: [DATE]
Resources available: [PEOPLE, BUDGET, TOOLS]
My experience with this type of task: [NONE / SOME / EXPERIENCED]

Help me:
1. Break this into the first 5 concrete steps
2. Identify the very first action I can take in the next 10 minutes
3. Create a rough timeline working backward from the deadline
4. List the information I need to gather before going further
5. Identify who I should talk to for guidance
6. Define "done" — what does success look like?
```

## System Maintenance Checklist

Run this monthly to keep your productivity system effective.

```
MONTHLY SYSTEM MAINTENANCE

[ ] Review and update my Personal Context Prompt
    (new projects, changed goals, updated team)
[ ] Archive prompts I no longer use
[ ] Refine my 3 most-used prompts based on experience
[ ] Update my quarterly goals if they have changed
[ ] Review my prompt library for outdated information
[ ] Add any new prompts I created ad-hoc this month
[ ] Check if my time estimates are accurate
    (am I spending too long on AI interactions?)
[ ] Share improvements with my team
[ ] Back up my prompt library
[ ] Set a calendar reminder for next month's maintenance
```
