## Platform 1: LinkedIn

### LinkedIn Post — Thought Leadership

```
Write a LinkedIn post about [TOPIC].

My professional context:
- Role: [TITLE]
- Industry: [SECTOR]
- Years of experience: [NUMBER]
- Audience: [WHO FOLLOWS ME]

Post style: [SELECT ONE]
A) Personal story with professional lesson
B) Contrarian opinion backed by evidence
C) Framework or model I use
D) Lessons from a failure
E) Industry trend analysis
F) Celebrating a team/colleague achievement

Structure:
- Opening hook (first line — this appears before "...see more")
  [Must stop the scroll — use a bold statement, surprising number, or question]
- Body (short paragraphs, 1-2 sentences each)
- Key insight or lesson
- Closing CTA (question that drives comments)
- 3-5 hashtags

Length: [SHORT (under 500 chars) / MEDIUM (500-1300 chars) / LONG (1300-3000 chars)]
Tone: [PROFESSIONAL / CONVERSATIONAL / INSPIRATIONAL / PROVOCATIVE]

Write 3 versions with different hooks. Rank by likely engagement.
```

### LinkedIn Post — Engagement / Poll

```
Create a LinkedIn engagement post with a poll or discussion prompt.

Topic: [SUBJECT AREA]
My audience: [DESCRIPTION]
Goal: [ENGAGEMENT / BRAND AWARENESS / THOUGHT LEADERSHIP]

Create:
1. A poll question with 4 options:
   - Question: [Something people have strong opinions about]
   - Option A: [CHOICE]
   - Option B: [CHOICE]
   - Option C: [CHOICE]
   - Option D: Other (comment below)

2. The post text to accompany the poll:
   - Context for why this question matters
   - Your own perspective (to model engagement)
   - Invitation to elaborate in comments

3. A follow-up comment to post after getting initial responses
   (to boost engagement and keep the conversation going)
```

### LinkedIn Article (Long-Form)

```
Write a LinkedIn article about [TOPIC].

Target keyword/search term: [WHAT PEOPLE WOULD SEARCH]
Target reader: [WHO AND WHAT THEY CARE ABOUT]
My unique angle: [WHAT MAKES MY PERSPECTIVE DIFFERENT]
Length: 800-1,500 words

Structure:
1. Headline (SEO-friendly, curiosity-driving)
2. Opening paragraph (hook with a story, statistic, or bold claim)
3. The Problem (what most people get wrong about this topic)
4. My Framework/Approach (3-5 key points with subheadings)
5. Evidence (examples, data, case studies)
6. Practical Application (what the reader can do TODAY)
7. Conclusion (circle back to the opening, reinforced by the argument)
8. CTA (follow, comment, or visit link)

Include:
- At least 2 personal anecdotes or experiences
- At least 3 data points or external references
- Subheadings every 200-250 words
- A "tweetable" pull quote for easy sharing
```

### LinkedIn Carousel (Document Post)

```
Create a LinkedIn carousel post on [TOPIC].

Number of slides: [8-12]
Visual style: [CLEAN / BOLD / CORPORATE / CREATIVE]
Brand colors: [HEX CODES OR DESCRIPTIONS]

For each slide:
- Slide number
- Headline (large text, 3-7 words)
- Body text (1-2 short sentences, max 30 words)
- Visual element suggestion (icon, chart, illustration)
- Design notes (text alignment, emphasis)

Carousel flow:
Slide 1: Cover — Hook headline + "Swipe for [BENEFIT]"
Slide 2: Context/Problem — Why this matters
Slides 3-[N-1]: Content — One key point per slide
Slide [N]: CTA — "Follow [HANDLE] for more [TOPIC]"
            + "Like + Save if this was helpful"
            + "Share with someone who needs this"

Also write the caption that accompanies the carousel:
- Hook line
- Brief context
- CTA
- Hashtags (5-7)
```

### LinkedIn Comment Strategy

```
Generate 10 thoughtful comments I can post on other people's LinkedIn content.

My expertise area: [TOPIC]
My professional perspective: [BRIEF DESCRIPTION]
Type of posts I engage with: [INDUSTRY LEADERS / PEERS / POTENTIAL CLIENTS]

For each comment:
1. The type of post it responds to (describe the hypothetical post)
2. The comment (3-5 sentences)
3. Why this comment adds value (not just "Great post!")

Comment strategies to include:
- Adding a contrasting perspective
- Sharing a relevant personal experience
- Asking a thought-provoking follow-up question
- Providing additional data or context
- Connecting the topic to a broader trend
- Sharing a practical tip related to the post
- Tagging someone who would find it valuable (suggest the type of person)
- Respectfully disagreeing with reasoning
- Amplifying with a real-world example
- Building on the post with a complementary framework
```

## Platform 2: X (Twitter)

### Single Tweet

```
Write a tweet about [TOPIC].

Angle: [INFORMATIVE / HUMOROUS / PROVOCATIVE / INSPIRING / PRACTICAL]
Target audience: [WHO]
Goal: [ENGAGEMENT / CLICKS / FOLLOWERS / BRAND AWARENESS]

Write 5 variations:
1. Question format (drives replies)
2. Bold statement format (drives retweets)
3. Listicle format (drives saves)
4. Hot take format (drives debate)
5. Practical tip format (drives saves and follows)

Rules:
- Under 280 characters each
- Front-load the value (first few words matter most)
- [INCLUDE / EXCLUDE] emojis
- [INCLUDE / EXCLUDE] hashtags (max 2)
- No links in the tweet (kills reach — put in reply)
```

### Twitter/X Thread

```
Write a Twitter/X thread about [TOPIC].

Thread length: [7-15] tweets
Audience: [DESCRIPTION]
Goal: [EDUCATE / ENTERTAIN / PERSUADE / BUILD AUTHORITY]
My voice: [DESCRIBE YOUR TWITTER PERSONA]

Thread structure:
Tweet 1: HOOK — The most important tweet. Must make people click "Show this thread."
  Options:
  a) "I spent [TIME] studying [TOPIC]. Here is what I learned:"
  b) "[BOLD CLAIM] Here is why (thread):"
  c) "[NUMBER] [THINGS] about [TOPIC] that most people get wrong:"
  d) "Stop doing [COMMON MISTAKE]. Do this instead (thread):"

Tweets 2-[N-1]: BODY — One clear point per tweet
  - Each tweet should be valuable standalone
  - Use examples, data, or stories
  - Vary the format (some tweets are text, some suggest images)

Tweet [N]: CLOSE
  - Summarize in 1-2 sentences
  - CTA: Follow @[handle] for more [topic]
  - Ask for retweet: "RT tweet 1 if this was helpful"

Format each tweet with the number: "1/"
Ensure each tweet is under 280 characters.
```

### Twitter/X Reply Strategy

```
Generate replies for common engagement opportunities on Twitter/X.

My niche: [TOPIC]
My handle: @[HANDLE]
My personality on Twitter: [WITTY / HELPFUL / ANALYTICAL / BOLD]

Create template replies for:
1. When someone asks a question about [TOPIC]
2. When an industry leader posts an opinion I agree with
3. When someone shares a win related to my field
4. When someone posts a common misconception
5. When a trending topic intersects with my expertise
6. When someone tags me or mentions my content
7. When someone shares my content (quote tweet response)

Each reply should:
- Add value (not just "I agree")
- Be under 280 characters
- Show personality
- Invite further conversation
```

## Platform 3: Instagram

### Instagram Feed Post Caption

```
Write an Instagram caption for a [POST TYPE: image / carousel / reel] post.

Account type: [PERSONAL BRAND / BUSINESS / CREATOR]
Niche: [TOPIC/INDUSTRY]
Visual description: [WHAT THE IMAGE OR VIDEO SHOWS]
Tone: [CASUAL / POLISHED / QUIRKY / INSPIRATIONAL / EDUCATIONAL]

Caption structure:
1. HOOK (first line — appears before "...more")
   Write 3 hook options:
   a) Question hook
   b) Statement hook
   c) Story hook

2. BODY
   Style: [SELECT]
   a) Story format (personal narrative with lesson)
   b) Tips format (numbered list of advice)
   c) Behind-the-scenes (authentic, raw insight)
   d) Value-first (teach something useful)

   Length: [SHORT (1-3 sentences) / MEDIUM (paragraph) / LONG (storytelling)]

3. CTA (call to action)
   Options:
   a) "Save this for later"
   b) "Tag someone who needs to hear this"
   c) "Drop a [EMOJI] if you agree"
   d) "Comment [WORD] and I will send you [RESOURCE]"
   e) "Link in bio for [RESOURCE]"
   f) "What would you add to this list?"

4. HASHTAGS (25-30, organized by size)
   - 5 large hashtags (1M+ posts) for discovery
   - 10 medium hashtags (100K-1M) for competition
   - 10 small/niche hashtags (10K-100K) for ranking
   - 5 micro hashtags (under 10K) for community
```

## Platform 4: TikTok

### TikTok Video Script

```
Write a TikTok script about [TOPIC].

Duration: [15 / 30 / 60] seconds
Content style: [SELECT ONE]
a) Educational ("Did you know..." / "How to...")
b) Storytelling ("The time I..." / "A [PROFESSION] reveals...")
c) Trend-based (using trending sound/format)
d) POV format ("POV: You are...")
e) List format ("3 things you should never do when...")
f) Before/After transformation
g) Day in the life
h) Response to a comment/question

Creator type: [BUSINESS / PERSONAL BRAND / EDUCATOR / ENTERTAINER]
Target audience: [AGE GROUP AND INTERESTS]

Script:
[SECOND] | [ACTION/VISUAL] | [SPOKEN WORDS] | [ON-SCREEN TEXT]

00-01 | [HOOK visual — must stop the scroll] | [First words] | [Text overlay]
01-03 | [Setup] | [Context] | [Text]
03-[END-5] | [Main content] | [Core message] | [Key text]
[END-5]-[END] | [Closing] | [CTA] | [Final text]

Also provide:
- 3 hook options (the first 1-3 seconds determine everything)
- Sound/music recommendation (vibe description)
- Caption (under 150 characters + CTA)
- 5 hashtags (mix of niche and trending)
- Comment to pin: [FIRST COMMENT FOR ENGAGEMENT]
- Reply-bait: [QUESTION DESIGNED TO GET COMMENTS]
```

### TikTok Content Calendar

```
Create a 7-day TikTok content plan.

Account niche: [TOPIC]
Posting frequency: [1-3 POSTS PER DAY]
Content pillars: [3-4 RECURRING THEMES]
Current follower count: [NUMBER]
Growth goal: [WHAT WE ARE TRYING TO ACHIEVE]

For each day, provide:
- Post concept (one sentence description)
- Content type (educational / entertaining / trending / personal)
- Hook (first 2 seconds)
- Full script outline
- Estimated production time
- Sound/music type
- Best posting time
- Hashtags (5)

Content mix should be:
- 40% Educational (value-first content)
- 30% Entertaining (personality, humor, relatability)
- 20% Trending (trending sounds and formats adapted to your niche)
- 10% Promotional (subtle, value-wrapped promotions)
```

```
I have this content idea: [DESCRIBE YOUR CORE IDEA OR PASTE A BLOG POST]

Repurpose it for each platform with native formatting:

1. LINKEDIN POST
   [Full post with hook, body, CTA, hashtags]

2. TWITTER/X THREAD
   [7-10 tweet thread with numbering]

3. INSTAGRAM CAROUSEL
   [8-10 slides with headline and body text per slide + caption]

4. TIKTOK SCRIPT
   [30-second script with visual/audio/text breakdown]

5. YOUTUBE SHORT
   [60-second script with timestamps]

Each version should:
- Feel native to the platform (not just a copy-paste)
- Adjust tone for the platform's culture
- Use platform-specific CTAs
- Optimize for the platform's algorithm preferences

Also provide a posting schedule:
Which platform to post on which day for maximum reach without audience fatigue.
```

### Content Performance Analysis Prompt

```
Analyze the performance of my recent social media content:

Platform: [PLATFORM]
Posts analyzed: [NUMBER]
Time period: [DATES]

Performance data:
Post 1: [TOPIC] — [IMPRESSIONS] — [ENGAGEMENT] — [SAVES] — [SHARES] — [COMMENTS]
Post 2: [SAME FORMAT]
[CONTINUE FOR ALL POSTS]

Please analyze:
1. Which post types performed best and worst?
2. Are there patterns in topic, format, or posting time?
3. What engagement metrics suggest about content quality?
4. Hypotheses for why top performers worked
5. Hypotheses for why bottom performers underperformed
6. Specific recommendations for next week's content
7. Optimal posting schedule based on engagement patterns
8. Content types to create more of / less of
```

## Hashtag Strategy

### Hashtag Research Prompt

```
Develop a hashtag strategy for my [PLATFORM] account.

Niche: [TOPIC]
Target audience: [DESCRIPTION]
Competitors: [LIST 3-5 ACCOUNTS]
Location focus: [GLOBAL / NATIONAL / LOCAL]

Create:
1. Core hashtags (always use): [10 hashtags]
2. Niche hashtags (use for specific topics): [15 hashtags per topic]
3. Community hashtags (engage with community): [10 hashtags]
4. Trending/seasonal hashtags (rotate monthly): [10 hashtags]
5. Branded hashtag suggestion: [1-2 unique to your brand]
6. Competitor hashtags to monitor: [10 hashtags]

For each hashtag, estimate:
- Relevance to my content (High/Medium/Low)
- Competition level (Low/Medium/High)
- Discovery potential (Low/Medium/High)

Organize into sets of 20-30 for easy copy-paste with different mixes.
```
